**CKAN is the world's leading open-source data portal platform.** CKAN makes it easy to publish, share and work with data. It's a data management system that provides a powerful platform for cataloging, storing and accessing datasets with a rich front-end, full API (for both data and catalog), visualization tools and more. Read more at [ckan.org](http://ckan.org/).

## Features

CKAN provides a rich set of features to publish, manage, and consume data.

| Feature | Description |
| :--- | :--- |
| **Data Visualization** | Create charts and graphs from dataset resources. The screenshot shows a line graph visualizing Ebola case data over time. |
|

 |
| **Dataset & Group Management** | Organize datasets into logical collections or groups. The "Most popular groups" snippet highlights active collections. |

 |
| **Usage Tracking** | A fire icon and tooltip indicate a dataset's popularity based on recent views, helping users discover relevant data. |

## Support

If you need help with CKAN or want to ask a question, use one of the following channels. Please search the archives first to see if your question has already been answered.

- **Mailing List:** [ckan-dev](https://groups.google.com/a/ckan.org/forum/#!forum/ckan-dev) for development, deployment, and general questions. ([Archives](https://groups.google.com/a/ckan.org/g/ckan-dev))
- **Community Chat:** [CKAN chat on Gitter](https://gitter.im/ckan/chat) for real-time discussions with developers and community members.
- **Stack Overflow:** Use the [`ckan` tag](http://stackoverflow.com/questions/tagged/ckan) for technical questions.

If you've found a bug, please search the [GitHub Issues](https://github.com/ckan/ckan/issues) first, then open a new issue if it's not already reported.

If you find a potential security vulnerability, please email **security@ckan.org** rather than creating a public issue.

## Contributing to CKAN

For information on contributing to CKAN or its documentation, see [CONTRIBUTING.md](https://github.com/ckan/ckan/blob/master/CONTRIBUTING.md).

The CKAN community collaborates on the following platforms:

- **Mailing List:** Subscribe to the [ckan-dev](https://groups.google.com/a/ckan.org/forum/#!forum/ckan-dev) mailing list to receive news and participate in discussions about CKAN development and deployment.
- **Community Chat:** Join the public [CKAN chat on Gitter](https://gitter.im/ckan/chat) to talk with developers and other community members.
- **Wiki:** If you've figured out how to do something with CKAN and want to document it, add a page to the [CKAN wiki](https://github.com/ckan/ckan/wiki).
