**CKANは、世界をリードするオープンソースのデータポータルプラットフォームです。** CKANを使用すると、データの公開、共有、活用が簡単になります。豊富なフロントエンド、完全なAPI（データおよびカタログ用）、可視化ツールなどを備え、データセットのカタログ化、保存、アクセスを行うための強力なプラットフォームを提供するデータ管理システムです。詳細は [ckan.org](http://ckan.org/) をご覧ください。

## 機能

CKANは、データの公開、管理、利用に必要な豊富な機能を提供します。

| 機能 | 説明 |
| :--- | :--- |
| **データ可視化** | データセットのリソースからチャートやグラフを作成できます。スクリーンショットは、エボラ出血熱の症例データの経時変化を可視化した折れ線グラフです。 |
| 
![エボラ出血熱の症例データの経時変化を可視化した折れ線グラフ。](views_overview.png)
 |
| **データセットとグループの管理** | データセットを論理的なコレクションやグループに整理できます。「Most popular groups（最も人気のあるグループ）」のスニペットは、アクティブなコレクションをハイライト表示します。 |
| 
![さまざまなデータセットコレクションを紹介する「Most popular groups」セクション。](most-popular-groups-snippet.png)
 |
| **利用状況のトラッキング** | 炎のアイコンとツールチップで、最近の閲覧数に基づいたデータセットの人気度を示し、ユーザーが関連データを見つけやすくします。 |
| 
![データセットのタイトルの横にある炎のアイコンは、その人気度を示しています。](popular-dataset.png)
 |
| **カスタマイズ** | 管理インターフェースから直接カスタムCSSを追加することで、CKANインスタンスの見た目や操作感を簡単にカスタマイズできます。 |
| 
![カスタムダークテーマが適用されたCKANのヘッダー。](custom-css.png)
 |

## サポート

CKANに関するサポートが必要な場合や質問がある場合は、以下のいずれかのチャンネルをご利用ください。質問する前に、すでに回答がないかアーカイブを検索してください。

- **メーリングリスト:** 開発、デプロイ、一般的な質問については [ckan-dev](https://groups.google.com/a/ckan.org/forum/#!forum/ckan-dev) をご利用ください。（[アーカイブ](https://groups.google.com/a/ckan.org/g/ckan-dev)）
- **コミュニティチャット:** 開発者やコミュニティメンバーとのリアルタイムなディスカッションには [CKAN chat on Gitter](https://gitter.im/ckan/chat) をご利用ください。
- **Stack Overflow:** 技術的な質問には [`ckan` タグ](http://stackoverflow.com/questions/tagged/ckan) を使用してください。

バグを見つけた場合は、まず [GitHub Issues](https://github.com/ckan/ckan/issues) を検索し、まだ報告されていない場合は新しいIssueを作成してください。

潜在的なセキュリティ脆弱性を見つけた場合は、公開のIssueを作成するのではなく、**security@ckan.org** 宛てにメールでご報告ください。

## CKANへの貢献

CKANやそのドキュメントへの貢献に関する情報については、[CONTRIBUTING.md](https://github.com/ckan/ckan/blob/master/CONTRIBUTING.md) を参照してください。

CKANコミュニティは以下のプラットフォームで協力しています：

- **メーリングリスト:** [ckan-dev](https://groups.google.com/a/ckan.org/forum/#!forum/ckan-dev) メーリングリストに登録して、ニュースを受け取ったり、CKANの開発やデプロイに関するディスカッションに参加したりできます。
- **コミュニティチャット:** 公開されている [CKAN chat on Gitter](https://gitter.im/ckan/chat) に参加して、開発者や他のコミュニティメンバーと交流できます。
- **Wiki:** CKANで何かを実現する方法を見つけ、それをドキュメント化したい場合は、[CKAN wiki](https://github.com/ckan/ckan/wiki) にページを追加してください。
